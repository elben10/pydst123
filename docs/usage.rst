==============
Usage of PYDST
==============

Make sure that you've installed :code:`pydst`. You can verify that pydst is installed
simple by running in your favorite command line [#f1]_:

    .. code-block:: bash

        python -m pydst --version

    .. rubric:: footnotes

    .. [#f1] Assuming python is added to the path.
